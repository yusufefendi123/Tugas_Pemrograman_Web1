Praktikum Pemograman Web

Praktikum Pemograman Web memberikan Keterampilan kepada mahasiswa dalam pembuatan situs web yang dinamis dan dapat berinteraksi denan user. Materi meliputi : Teknologi Web, Pembuatan situs, HTML,CSS,PHP,MySQL dan Visual Studio Code..

Keterangan:

1. Buatlah class bersama .dashed untuk heading utama

2. Gunakan selector div untuk membuat Box Model

3. Buatlah class bernama underline-red untuk kalimat dengan garis bawah

4. Buatlah class bernama shadowuntuk memberikan efek bayangan pada teks
